    • Cara Penggunaan

    Dalam menggunakan robot line follower kita harus memperhatikan beberapa hal, seperti:

    a. Persiapan jalur

    Sebelum menggunakan robot line follower, pastikan jalur yang akan digunakan sudah disiapkan terlebih dahulu. Biasanya jalur menggunakan garis hitam di atas permukaan warna kontras seperti putih. Lalu, pastikan jalur datar dan bersih agar sensor bisa membaca garis dengan baik.

    b. Pastikan robot dalam kondisi baik

    Sebelum digunakan, periksa keadaan robot terlebih dahulu. Pastikan sensor dan roda tidak kotor atau terhalang.

    c. Kalibrasi sensor

    Beberapa robot line follower mungkin perlu dikalibrasi sensornya agar presisi mengikuti garis. Biasanya ada trimmer atau potensiometer untuk menyetel sensitivitas sensor.

    d. Nyalakan robot line follower

    Hidupkan robot line follower sesuai petunjuk.

    e. Letakkan robot di awal jalur

    Posisikan robot line follower tepat di awal jalur, biasanya di bagian garis hitam.

    f. Jalankan robot

    Beberapa robot mungkin memiliki tombol play atau maju, tergantung modelnya.
